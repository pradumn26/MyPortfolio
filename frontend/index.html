Emaily
Emaily is a survey creation web app through which users can create and send simple surveys to any number of users via email. This app uses oauth user authentication for creation and handling of user accounts. A user can then have to add some credits to his/her account in order to create surveys. This can be done by dummy credit card option which uses stripe api. Once the user have enough credits he/she can then create a new survey by filling a form. The user can then send survey via email which is powered by the sendgrid api.

Pictogram
Pictogram is basically a instagram clone. A user can directly signup to pictogram by filling a form or he/she can link their facebook or google account with pictogram. Like in instagram, a user can upload pictures, like, comment and follow other users. It also have notification feature, which displays all the current notifications of user read or unread. It consumes libraries like passportjs, multerjs, expressjs etc.
